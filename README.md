# 🧮 Quadratic Equation Solver

A simple Streamlit web app to solve any quadratic equation of the form:

\[
ax^2 + bx + c = 0
\]

---

## 🚀 Features

- Input coefficients: a, b, and c
- Calculates the discriminant (Δ)
- Handles:
  - Two real solutions
  - One real repeated solution
  - Two complex solutions
